# Backend server

A template Django server.

1. Install dependencies: `poetry install` and activate the environment if not in dev container
2. Run: `python manage.py runserver`

## Styling
This repository uses Flake8 to enforce styling. 
To check if your branch complies, you can run the `flake8` command. 
It's a good idea to run this before you make commits.
